# 🚀 SAM Quick Start Guide

**Get SAM running in 3 simple steps!**

## ⚡ Quick Installation

### Step 1: Download SAM
```bash
git clone https://github.com/forge-1825/SAM.git
cd SAM
```

### Step 2: Run Setup
```bash
python setup_sam.py
```

### Step 3: Start SAM
```bash
python start_sam_simple.py
```
*Or use the advanced launcher:*
```bash
python start_sam.py
```

**That's it!** 🎉 SAM will open in your browser at `http://localhost:8502`

## 🆘 Need Help?

### Common Issues

**"Streamlit not found"**
```bash
pip install streamlit
python setup_sam.py
```

**"Permission denied"**
```bash
# On macOS/Linux:
chmod +x setup_sam.py
python setup_sam.py
```

**"Python version error"**
- Install Python 3.8+ from [python.org](https://python.org)
- Try: `python3 setup_sam.py`

### Manual Installation

If the quick setup doesn't work:

```bash
# Install dependencies
pip install streamlit requests

# Create directories
mkdir -p security logs data cache

# Start SAM
python secure_streamlit_app.py
```
